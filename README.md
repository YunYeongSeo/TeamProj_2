# 🏭 생산라인 모니터링 채팅 프로그램 (TeamProj_2)

생산 현장의 상태를 **실시간으로 모니터링**하고,  
관리자와 작업자 간 **즉각적인 커뮤니케이션**이 가능하도록 개발한  
**생산라인 모니터링 · 채팅 통합 데스크톱 프로그램**입니다.

---

## 📌 프로젝트 개요 (Overview)

- 생산라인의 **영상 스트리밍, 센서 데이터, 채팅 기능**을 하나의 프로그램으로 통합
- 현장 상황을 실시간으로 파악하고, 문제 발생 시 즉각적인 대응이 가능하도록 설계
- 팀 프로젝트를 통해 **역할 분담 기반 개발 및 기능 통합**을 경험

---

## 👥 프로젝트 정보

- **프로젝트 유형** : 팀 프로젝트
- **팀 인원** : 5명
- **개발 환경**
  - Language : C++ 
  - Framework : MFC
  - DB : MySQL
  - Server : Python (Flask)
  - Embedded : Raspberry  Pi 4
  - IDE : Visual Studio, Visual Studio Code

---

## 🛠️ 주요 기능 (Features)

- 생산라인 **실시간 상태 모니터링**
- 관리자 ↔ 작업자 **실시간 채팅 기능**
- 센서 데이터 수집 및 UI 표시
- 사용자 권한에 따른 화면 및 기능 분리
- 시스템 상태 로그 출력

---

## 👩‍💻 담당 역할 (My Role)

생산라인 모니터링 채팅 프로그램에서  
**MFC 기반 관리자 UI 및 상태 정보 표시 화면 구현**을 담당했습니다.

- **MFC 기반 생산라인 모니터링 UI 설계 및 구현**
  - 관리자 화면을 중심으로 한 생산라인 모니터링 UI 구성
  - 실시간으로 수집되는 정보가 직관적으로 보이도록 화면 레이아웃 설계

- **검사·상태 정보 시각화 화면 구현**
  - 센서 데이터 및 시스템 상태 정보를 MFC UI에 표시
  - 로그 메시지 영역을 구성하여 시스템 상태 변화 확인 가능하도록 구현

- **관리자 / 직원 권한 분리 구조 이해 및 UI 동작 검증**
  - 사용자 권한에 따른 화면 접근 흐름 정리
  - 직원 화면은 권한 제한 동작 검증용으로 분리 구현

- **팀 프로젝트 협업 및 기능 통합**
  - 팀원과의 역할 분담을 기반으로 기능 구현
  - UI와 서버·센서 데이터 연동 과정에서 발생한 오류 수정 및 통합 작업 참여
