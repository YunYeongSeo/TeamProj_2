barcode_detection_log : 카메라에서 검출된 바코드의 메타데이터(바운딩박스, 신뢰도, 이미지 경로/파일명, 검출시각)를 사건 단위로 기록하는 로그.

chat_event : 클라이언트별 메시지/알림/상태 같은 채팅 이벤트 스트림을 시각순으로 적재하는 이벤트 로그.

chat_message : 발신 사번과 본문을 가진 일반 채팅 메시지 히스토리(검색·정렬용 인덱스 포함).

environment_logs : 온도/습도를 주기·수동·기동 시점별로 수집한 환경 센서 로그.

line_metric : 라인ID·지표명·값을 시계열로 저장하는 생산 라인 성능/상태 메트릭 테이블.

login_history : 사번·세션ID 기준으로 로그인/로그아웃 결과·기간·클라이언트 정보를 남기는 추적 로그.

notifications : 유형/심각도/읽음 상태/이미지/부가 JSON 데이터를 포함하는 통합 알림 피드(큐).

user_account : 사번 기반 사용자 계정(비밀번호 해시·권한·활성 여부) 관리.

user_session : 현재/최근의 세션 상태(활성·최근활동·임시 여부) 를 추적하는 실시간 세션 테이블.
